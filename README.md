# Responsive Frontend Blog

A modern, responsive blog application built with **HTML, CSS, JavaScript, and Bootstrap**. Users can **add, edit, and delete posts entirely on the frontend**. Portfolio-ready project demonstrating UI/UX and frontend development skills.


## Features
- Add new posts with title and content
- Edit existing posts
- Delete posts with confirmation modal
- Responsive design (desktop & mobile)
- Modern UI using Bootstrap 5



## Project Structure
```
frontend-blog/
├── index.html
├── script.js
├── styles.css
└── README.md
```

- CSS3
- JavaScript (ES6)
- Bootstrap 5
